# üçî Cravecraft- The ultimate Recipe Finder Web application

CraveCraft is a responsive web application that helps its users find meals by using ingredients they already possess. The motivation for this project was to develop a fluid, mobile-first cooking companion that would provide instant and intuitive recipe finding.

![Project Preview](https://via.placeholder.com/800x400?text=App+Screenshot+Coming+Soon)

## Overview of the Project -
This project's aim was to design a practical, user-friendly interface that actually interacts with a real-life database. It solves the common problem of "what to cook" by providing detailed instructions, ingredient measures, and video tutorials in one view. 

## ‚ú® Key Features
-**Search Engine:** Fetching of recipes by name or main ingredient in real time.
-**Responsive Design:** Optimized layout that adapts perfectly from mobile phones to desktops - using Tailwind CSS.
- **Recipe Details:** Complete breakdown of ingredients, measurements, and step-by-step instructions.
- **Video Integration:** Embedded YouTube tutorials to cater for visual learners.
-**Favorites Manager:** Users can favorite recipes for later access. Data is persisted locally; favorites remain saved even after closing the browser.


## Tech Stack
-**Framework:** React.js + Vite
- **Styling:** Tailwind CSS
- **State Management:** Context API

- **Routing:** React Router DOM

- **API:** TheMealDB
